

## Options present to you when you are deciding to merge pull request (available to reviewers, etc)
- Create a merge commit (all commits will be merged with main)
	  - retains history of all commits on the head branch
- Squash and merge (all commits will be squashed into one commit, and added to main as a new single commit)
- Rebase and merge (1 commit will be added and rebased)
